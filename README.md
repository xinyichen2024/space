# 引力之眼 · H5 星空观测实验室

一个无需构建工具的交互式 H5 页面，用可视化方式展示不同质量天体对背景星系的引力透镜影响，并提供焦距与远场成像分析。

## 主要功能

- 程序化深空背景，包含旋涡、棒旋、椭圆和侧向盘状星系
- 质量范围覆盖 1—10⁸ 个太阳质量
- 天体拖放、自动漂移、暂停、选中和删除
- 扩展光源逐点薄透镜成像，可呈现主像、次像、切向弧与爱因斯坦环
- 恒星、中子星、恒星级黑洞和超大质量黑洞预设
- 独立的成像清晰度与最佳观察位置分析工作台

## 运行

直接打开 `index.html`，或在当前目录启动任意静态文件服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 星空交互

- 从右侧面板将天体拖入星空，单击卡片也可快速放置
- 天体进入视场后会自动漂移并在观测边界反弹
- 顶部按钮可暂停或继续全部天体运动
- 拖动画面内的天体调整位置，释放后会继续自动运动
- 单击天体选中，然后点击“移出视场”或按 `Delete`
- 双击天体可直接删除

## 成像清晰度分析

点击右侧面板中的“成像分析”进入工作台。若主界面已有选中天体，质量和结构模型会自动带入。

可调参数包括：

- 透镜体质量与结构模型
- 源星系距离
- 透镜—观察者距离
- 源、透镜和观察者之间的光轴偏差

工作台对恒星、中子星和黑洞统一计算“成像清晰度最优位置”。三种模型使用同一套虚拟观测设备参数：

- 角分辨率为 1 mas
- 以爱因斯坦环角半径 0.1″ 作为最适合当前画布采样和展示细节的目标尺度
- 清晰度评分共同考虑观察距离偏差、角分辨率和光轴对准误差
- “移动到最清晰位置”会把观察者移到清晰度函数的峰值位置

远场薄透镜的爱因斯坦角半径为：

```text
θE² = (4GM/c²) × DLS / (DL × DS)
```

程序令 θE 达到统一的 0.1″ 目标尺度并反求观察距离。因此，在质量、源距离和对准条件相同时，三种天体首先得到相同的远场成像位置；结构差异只作为额外约束。

### 恒星模型

恒星具有不透明光球。表面掠射光线的焦线起点为：

```text
f = b²c² / 4GM
```

如果统一求得的成像位置落在焦线起点以内，程序会把最优位置外移到焦线起点，避免把被恒星表面遮挡的光路计入清晰成像。恒星自身的耀光与日冕噪声尚未纳入评分。

### 中子星与黑洞模型

中子星和黑洞没有采用公里级的弱场“表面焦距”。二者在远场与恒星共用相同的爱因斯坦环和清晰度求解；近场偏折角仍标记为需要完整广义相对论光线追迹。

页面同时检查结构模型与质量是否合理：

- 主序恒星演示范围约为 0.08–150 M☉
- 中子星演示范围约为 1–3 M☉
- 低于约 3 M☉ 的黑洞模型会显示提示
- 超出范围时保留数学结果，但明确提示应更换结构模型

结果区还保留焦线起点或 1 mas 环可分辨距离上限作为辅助数据，它们不再被称为“最佳观测距离”。

## 文件结构

- `index.html`：主界面与成像清晰度分析工作台结构
- `styles.css`：星空及分析界面的视觉样式
- `app.js`：星空、天体运动和扩展光源透镜渲染
- `focus.js`：最佳观察位置、观察几何和成像清晰度计算

## 模型说明

本项目用于科普展示。星系成像采用简化薄透镜模型，恒星焦线采用弱场近似，视觉尺度经过对数映射。中子星和黑洞近场未进行完整 Schwarzschild/Kerr 时空测地线积分，结果不用于精密天文计算、航天器轨迹规划或实际观测任务设计。
