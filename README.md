# 引力之眼 · H5 星空观测实验室

一个无需构建工具的交互式 H5 页面，用可视化方式展示不同质量天体对背景星空的引力透镜影响。

## 运行

直接打开 `index.html`，或在当前目录启动任意静态文件服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 交互

- 从右侧面板将天体拖入星空（单击卡片也可快速放置）
- 天体进入视场后会自动漂移并在观测边界反弹；顶部可以暂停或继续运动
- 拖动画面内的天体改变位置，释放后会继续自动运动
- 单击天体选中，然后点击“移出视场”或按 `Delete`；也可以直接双击删除
- 自定义质量范围为 1—10⁸ 个太阳质量

说明：本项目是用于科普展示的“物理启发式”实时模拟。星光偏折、双像和爱因斯坦环基于简化薄透镜模型，视觉尺度经过对数映射，并非用于精密天文计算。
