# 引力之眼 · H5 星空观测实验室

一个无需构建工具的交互式 H5 页面，用可视化方式展示不同质量天体对背景星系的引力透镜影响，并提供焦距与远场成像分析。

## 主要功能

- 程序化深空背景，包含旋涡、棒旋、椭圆和侧向盘状星系
- 质量范围覆盖 1—10⁸ 个太阳质量
- 天体拖放、自动漂移、暂停、选中和删除
- 扩展光源逐点薄透镜成像，可呈现主像、次像、切向弧与爱因斯坦环
- 恒星、中子星、恒星级黑洞和超大质量黑洞预设
- 独立的焦距与观察距离分析工作台

## 运行

直接打开 `index.html`，或在当前目录启动任意静态文件服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 星空交互

- 从右侧面板将天体拖入星空，单击卡片也可快速放置
- 天体进入视场后会自动漂移并在观测边界反弹
- 顶部按钮可暂停或继续全部天体运动
- 拖动画面内的天体调整位置，释放后会继续自动运动
- 单击天体选中，然后点击“移出视场”或按 `Delete`
- 双击天体可直接删除

## 焦距分析

点击右侧面板中的“焦距分析”进入工作台。若主界面已有选中天体，质量和结构模型会自动带入。

可调参数包括：

- 透镜体质量与结构模型
- 源星系距离
- 透镜—观察者距离
- 源、透镜和观察者之间的光轴偏差

工作台实时展示目标星系 GLX–042 的光路、观察面、爱因斯坦环、切向弧、角半径、物理环半径和响应曲线。

### 恒星表面模型

恒星采用弱场偏折近似。表面掠射光线的焦线起点为：

```text
f = b²c² / 4GM
```

其中 `b` 为光球表面冲击参数。太阳质量与太阳半径对应的焦线起点约为 547.7 AU。引力透镜形成的是焦线，而不是普通光学透镜的唯一焦平面。

### 中子星与黑洞模型

中子星和黑洞近场属于强引力区，页面不会把弱场公式外推得到的公里级数值标为“最佳观测距离”。

致密天体使用远场薄透镜几何计算爱因斯坦角半径，并根据角分辨率阈值给出参考观察距离：

- 恒星级致密天体默认采用 1 mas 分辨阈值
- 超大质量黑洞默认采用 0.1″ 分辨阈值
- 成像可辨度由爱因斯坦角半径与光轴对准程度共同决定
- 表面掠射焦距和近场偏折角标记为需要完整广义相对论光线追迹

## 文件结构

- `index.html`：主界面与焦距分析工作台结构
- `styles.css`：星空及分析界面的视觉样式
- `app.js`：星空、天体运动和扩展光源透镜渲染
- `focus.js`：焦距、观察几何和成像可辨度计算

## 模型说明

本项目用于科普展示。星系成像采用简化薄透镜模型，恒星焦线采用弱场近似，视觉尺度经过对数映射。中子星和黑洞近场未进行完整 Schwarzschild/Kerr 时空测地线积分，结果不用于精密天文计算、航天器轨迹规划或实际观测任务设计。
